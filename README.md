# 단어 시험 (베트남어 ↔ 영어)

GYBM 베트남어 단어 시험·암기 웹앱입니다. **`단어시험.html` 파일 하나**로 동작하며, 브라우저에서 열면 바로 사용할 수 있어요 (인터넷 없이도 작동).

## 기능
- **주관식 / 객관식 / 플래시카드** 시험
- 방향 선택: 베트남어→영어 / 영어→베트남어 / 랜덤
- **단원별 선택** 또는 **번호로 범위 지정** (예: 1단원 30번 ~ 3단원 21번)
- 틀린 단어만 다시 풀기
- **인쇄용 시험지 만들기** (PDF 저장·여럿이 함께 풀기, 중복 자동 제거)
- 발음 듣기, 단어장 보기, 내 단어 추가

## 단어
- Lesson 1 ~ 12, 총 496개

## 여는 법
`단어시험.html`을 더블클릭하거나 브라우저로 열면 됩니다.
